<h1>Command Quest</h1>
<h3>Objectif</h3>
<p>Créer un framework pour développer de petits RPG multiplateforme rapidement.</p>
<h3>Fonctionnalités</h3>
<p>Un éditeur de jeu sera mis à disposition, ainsi que les outils nécessaires pour pouvoir déployer le projet sur plusisuers plateformes telles que Windows, Linux ou Android (if possible)</p>
